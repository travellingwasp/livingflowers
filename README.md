# WindowPlant Lab

Astro static website for a 30-day real-traffic experiment.

## Start

```bash
npm install
npm run dev
```

## Verify

```bash
npm run check
```

Before production deployment, set:

- `SITE_URL`
- `PUBLIC_CLOUDFLARE_WEB_ANALYTICS_TOKEN`

See `RUNBOOK.md` for Cloudflare Pages and Google Search Console setup.
