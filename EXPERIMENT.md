# WindowPlant Lab 30-Day Traffic Experiment

## Mission

Build and operate a small Astro website whose only objective is to obtain real, independently verifiable human traffic within 30 days.

Real traffic means:

- Organic clicks visible in Google Search Console.
- Verified browser visits in Cloudflare Web Analytics.
- Legitimate external referrers.
- No paid traffic, bots, traffic exchanges, purchased backlinks, or spam.

## Success Levels

Bronze:

- Site indexed.
- 100 Search Console impressions.
- 1 organic click.

Silver:

- 10 organic clicks.
- 25 verified human visits.
- 2 legitimate acquisition sources.

Gold:

- 100 organic clicks or 250 verified human visits.
- Meaningful traffic to more than one page.
- Evidence of repeatable acquisition.

## Stack

- Astro static site.
- GitHub repository as source of truth.
- Cloudflare Pages hosting.
- Cloudflare Web Analytics.
- Google Search Console.
- Markdown or Astro content, generated sitemap, robots.txt, RSS, canonical URLs, Open Graph metadata, responsive design, and minimal JavaScript.

## Owner-Gated Setup

The owner must provide or complete:

- Real production domain.
- Cloudflare Pages project connection.
- Cloudflare Web Analytics token.
- Google Search Console verification.
- Sitemap submission at `https://DOMAIN/sitemap.xml`.
